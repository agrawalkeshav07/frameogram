# FRAMEOGRAM

Premium optical shop website for FRAMEOGRAM, built with React, Vite, Tailwind CSS, and Framer Motion.

## Run Locally

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Deploy on Render

Use Render's Static Site service.

```text
Build Command: npm ci && npm run build
Publish Directory: dist
```

This repo also includes `render.yaml`, so Render can auto-detect the same settings as a Blueprint.

## Update Catalogue Content

Product categories, catalogue items, gallery items, and business details live in:

```text
src/data/siteData.js
```

Sample eyewear images and the FRAMEOGRAM logo live in:

```text
public/assets
```
